# OPI

**O**BS **P**ackage **I**nstaller

## System Requirements

- openSUSE Tumbleweed, openSUSE Leap 42.1+, SLE 12+
- perl
- perl-libwww-perl
- perl-XML-LibXML
- perl-URI
- perl-Config-Tiny

## Install

```
wget https://github.com/openSUSE-zh/opi/raw/master/opi
install opi ~/bin
```

## Use

```
opi pcsx2
```
